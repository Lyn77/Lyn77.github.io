一、基础


（1）html的表单是为了收集用户信息。
（2）表单通常由表单域、提示信息和表单控件（表单元素）组成。

二、input控件

  1. <input />为单标签，type属性为其最基本属性，还有其他属性。
  2. 表单属性


































































    属性 属性值 描述
    type text 文本框
    password 密码框
    radio 单选框
    checkbox 复选框
    button 按钮
    submit 提交按钮
    reset 重置按钮
    image 图像形式的提交按钮
    file 文件域
    name 自定义 控件名
    value 自定义 input控件的默认值
    size 正整数 input控件在页面中显示的宽度
    checked checked 默认被选中的项
    maxlength 正整数 控件允许输入的最多字符数
