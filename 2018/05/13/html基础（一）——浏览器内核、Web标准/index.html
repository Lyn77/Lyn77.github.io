二、Web标准

  1. 结构(structure):用于对网页元素进行整理和分类，主要包括XML和XHTML。
  2. 样式(Presentation):用于设置网页元素的板式、颜色、大小等外观样式，主要是CSS。
  3. 行为(Behavior):用于对网页模型的定义和交互的编写，主要包括DOM(Document Object Model)和ECMAscript(JavaScript)。
