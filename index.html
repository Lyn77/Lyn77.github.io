html基础（四）——路径

发表于 2018-05-13 | 阅读次数:
字数统计: 223 字 | 阅读时长 ≈ 1 分钟

说明：通常新建一个文件夹专门存放图像文件，如果插入图像，需要采用“路径”指定位置。

  1. 相对路径
    （1）图像和HTML文件名位于同一文件夹，只需输入图像文件的名称即可。如<img src=”logo.gif”/>。
    （2）图像文件位于HTML文件的下一级文件夹，这时需要输入文件夹名和文件名。如<img src=”img/img1.gif”/>。
    （3）图像文件位于HTML文件的上一级文件夹，在文件夹名称前加“../”。如<img src=”../logo.gif”/>。
  2. 绝对路径
    （1）点击图片→右键→位置 查询本地地址“c:\web\picture\logo.gif”（这种不用）。
    （2）在网络中点击图片→右击→复制图片地址。

html基础（三）——html标签

发表于 2018-05-13 | 阅读次数:
字数统计: 641 字 | 阅读时长 ≈ 3 分钟
  1. 标题标签
    (1)标题标签(head)
     <hn>标题文本</hn>
     注：h1标签一般用于logo，少用。
    (2)段落标签(paragraph)
     <p>段落文本<p>
     注：默认情况下，文本在段落中根据浏览器窗口大小自动换行。
    (3)水平线标签(horizontal)
     <hr />
    (4)换行标签(break)
     <br />
    (5)div span标签（没有语义）
     div是division的缩写，分割、分区。
     span为跨度、跨距、范围。
  2. 文本格式化标签


























    标签 含义 标准
    <b></b><strong></strong> 粗体 XHTML推荐使用stong
    <i></i><em></em> 斜体 XHTML推荐使用em
    <s></s><del></del> 加删除线 XHTML推荐使用del
    <u></u><ins></ins> 加下划线 XHTML不赞成使用u

     注：1）b、i、s、u只有使用没有强调的意思。
       2)strong、em、del、ins语义更强烈。
       3）标签属性<标签名 属性1=“属性值1” 属性2=“属性值2”>  key=”value”
  3. 图像标签(image)
     <img />
    1)<img src=” “ alt=” “ width=”400” border=”10”>
    2)<img />标签属性



































    属性 属性值 描述
    src url 图像的路径
    alt 文本 图像不能显示时替换的文本
    title 文本 鼠标悬停时现实的内容
    width 像素(XHTML不支持百分比) 设置图像的宽度
    height 像素(XHTML不支持百分比) 设置图像的高度
    border 数字 设置图像边框的宽度
  4. 链接标签(anchor)
    （1）<a>链接</a>
    （2）语法格式为:<a href=”跳转目标” target=”目标窗口的弹出方式”>文本或图像</a>
    注：
    1)href用于指定链接目标的地址，Hypertext Refrence的缩写，超文本引用。
    2)target用于指定页面的打开方式，取值有self（默认）和blank（新窗口）两种。
    3)如果当时没有确定链接目标，通常用#暂时表示为空链接。
    4)不仅可以创建文本链接，在网页中各种元素，如图像、表格、音频、视频等都可以添加超链接。
    （3）锚点定位
    通过创建锚点链接，用户能够快速定位到目标内容。
    1)使用<a href=”#id名”>链接文本</a>创建链接文本。
    2)使用相应的id名标注跳转目标的位置。
    （4）base标签
    <base />单标签
    base可以设置整体链接的打开方式。
  5. 特殊字符标签






























    特殊字符 描述 字符的代码
    空格 &nbsp;
    < 小于 &lt;
    > 大于 &gt;
    & 与 &amp;
    © 版权符号 &copy;
  6. 注释标签
    <!–注释语句–>

二、Web标准

  1. 结构(structure):用于对网页元素进行整理和分类，主要包括XML和XHTML。
  2. 样式(Presentation):用于设置网页元素的板式、颜色、大小等外观样式，主要是CSS。
  3. 行为(Behavior):用于对网页模型的定义和交互的编写，主要包括DOM(Document Object Model)和ECMAscript(JavaScript)。
